## 🎤 Introduce
단 한 명의 사용자라도 만족할 수 있는 서비스를 만드는 것을 목표로 합니다.

### 빠름보다 바름, 속력보다 방향

AI 시대에도 "There's no silver bullet"이라는 말처럼, 
문제 해결 과정에서 기초와 원리를 중요하게 생각합니다.

컴퓨터공학 전공으로 편입하여 자료구조, 운영체제 등 기본기를 보완했고, 자바 풀스택 부트캠프를 통해 Spring 기반 웹 서비스 개발과 팀 프로젝트 협업 경험을 쌓았습니다.

단순히 기능 구현에 그치지 않고, 데이터 흐름과 구조까지 이해하는 개발자가 되고자 합니다.
